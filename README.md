## Quick Start

To use sbt-github-download in an existing SBT project, add the following dependencies to your
`project/plugins.sbt` depending on your needs:

```scala
addSbtPlugin(
  "io.chrisdavenport" %% "sbt-github-download" % "<version>"
)
```
